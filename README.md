Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minha trajetória profissional, habilidades técnicas e projetos, utilizando uma interface interativa inspirada em editores de código e terminais de desenvolvimento.

## 🎯 Sobre o Projeto

O portfólio reflete meu momento atual: a transição de uma sólida experiência em logística e controle de estoque para a área de **Engenharia de Software**. A interface foi construída com um design *dark mode* elegante, onde as informações são apresentadas como arquivos de sistema (`.txt`, `.log`, `.json`), conectando minha paixão por tecnologia com a minha apresentação profissional.

### ✨ Funcionalidades Principais

* **Design Temático (IDE/Terminal):** Apresentação visual que simula um ambiente de desenvolvimento (arquivos `sobre_mim.txt`, `experiencia.log`, `habilidades.json`).
* **Interatividade (Drag & Drop):** Sistema interativo de cartões (cards) que podem ser arrastados.
* **Internacionalização:** Suporte bilíngue intuitivo — ao arrastar os cartões, o usuário pode visualizar o conteúdo traduzido para o inglês.
* **Responsividade:** Layout adaptável para diferentes tamanhos de tela (Mobile, Tablet e Desktop).

## 🛠️ Tecnologias Utilizadas

As seguintes tecnologias e ferramentas foram empregadas na construção deste portfólio:

* **HTML5 & CSS3:** Estruturação semântica e estilização avançada (animações, grid/flexbox, paleta de cores *dark theme*).
* **JavaScript (React.js):** Lógica de interatividade dos cartões, manipulação de eventos de arrastar (drag) e renderização condicional dos idiomas.
* **Hospedagem:** Vercel.

## 📂 Estrutura de Informações

A navegação do portfólio é dividida logicamente nos seguintes blocos:

1.  **`sobre_mim.txt`**: Um resumo do meu perfil, destacando minha proatividade na criação de soluções eficientes e sistemas interativos para automação de processos.
2.  **`experiencia.log`**: Histórico profissional, com foco no desenvolvimento de sistemas internos, automação de relatórios, dashboards em tempo real e controle logístico.
3.  **`habilidades.json`**: Formação acadêmica em Engenharia de Software e o mapeamento das minhas hard skills (Python, HTML, CSS, Lógica de Programação) e soft skills (Perfil Analítico, Trabalho em Equipe).
